About Todo App

Source code

Welcome to Todo App. As the name suggests, this is an application that helps you record tasks you have/want to do. This way you have somewhere where you can reference for your planned activities, thus boosting your productivity.

Todo App is built using Flask, a Python web framework. The Frontend is mostly Bootstrap with some little custom styling.

The motivation to build this application is actually for personal use. But feel free to use it if you want to. There are no ads or promotions here. Simply create an account and start enjoying the app.

For any suggestions, comments, or complaints, please submit an issue here. If you do not have a Github account, you can email me at kitakayaloisa@gmail.com.

Also, if you wish to contribute to this project, you will be pleased to know that it is open source. So all you need to do is simply make a pull request to the app repository on Github.
